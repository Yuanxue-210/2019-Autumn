#  两个物体之间若能发生碰撞，需要什么先决条件
 * 都选中Collider组件
----
#  触发器函数（Trigger 与 Collider）有几种类型
## 三种：
 1. 双方都有刚体，都有触发器，碰撞时各自执行碰撞脚本
 2. 一方有刚体，另一方没有，有刚体的执行脚本
 3. 都没有刚体，碰撞时会直接穿过去
